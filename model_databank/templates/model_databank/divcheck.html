Get this model on your computer

Download Mercurial

You'll need to download Mercurial and its `hg` command to get the model files on your computer. Mercurial is a version-control system that allows you to record changes made to model files. Also, with Mercurial you can check out previous versions of the model. It can be downloaded from http://mercurial.selenic.com/downloads.

Mercurial Quick Start

Clone a model and push changes (run these commands from the command-line). 

$ hg clone http://hg-test.3di.lizard.net/hhnkipad
$ cd hhnkipad
$ (change model file(s))
$ hg commit -m 'My changes'
$ hg push

Step by step

Clone the model

$ hg clone http://hg-test.3di.lizard.net/hhnkipad
By using the clone command, the model file repository is downloaded to your computer. This can take awhile, depending on the size of the model and your internet connection speed. Downloading the repository allows you to make incremental changes to model files.

Change the model

You can change the model files with your favorite GIS tools. Just make sure you save the changed files with the same name in the same directory they came from.

Commit your changes

$ hg commit -m 'My changes'
After you have saved your changes to the model, you'll need to commit these changes to the repository. Add a commit message that concisely describes the changes you made.

Upload your changes

$ hg push
Upload your changes to the server by running the push command.
